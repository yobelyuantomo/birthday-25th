# 💌 Website Ucapan Ulang Tahun — Paulin (25th)

## Cara ganti foto dummy jadi foto asli

Semua foto ada di folder `assets/images/`. Tinggal **timpa file dengan nama yang sama** (format jpg/png sama-sama bisa, asal nama file & ekstensi cocok dengan yang ada di `index.html`, atau ubah juga referensinya di `index.html` kalau nama filenya beda):

| Nama file               | Dipakai untuk                          |
|--------------------------|------------------------------------------|
| `foto-kecil.jpg`         | Foto masa kecil Paulin                   |
| `potret-1.jpg` s/d `potret-3.jpg` | Halaman potret 1 (3 foto)        |
| `potret-4.jpg` s/d `potret-6.jpg` | Halaman potret 2 (3 foto)        |
| `pasangan-1.jpg` s/d `pasangan-5.jpg` | Foto berdua / dating (5 foto) |

**Tips ukuran foto** supaya web tetap ringan & cepat dibuka:
- Resize ke lebar maksimal ±1200–1500px
- Kompres ke JPEG quality 75–85 (pakai [squoosh.app](https://squoosh.app) gratis & gampang)
- Target ukuran per foto idealnya di bawah 300KB

## Cara coba di laptop sebelum upload

Tinggal buka `index.html` langsung di browser (double click), atau pakai live server kalau ada.

## Cara upload ke GitHub Pages

1. Buat repo baru di GitHub, misal `paulin-25th`
2. Upload semua isi folder ini (index.html, style.css, script.js, folder assets/) ke repo tsb
3. Masuk ke **Settings → Pages**, pilih branch `main` dan folder `/ (root)`, lalu Save
4. Tunggu 1-2 menit, website akan live di `https://username-kamu.github.io/paulin-25th/`

## Cara pakai websitenya

- Klik di mana saja / tombol panah di bawah untuk lanjut ke halaman berikutnya
- Di HP bisa juga swipe ke atas untuk lanjut
- Halaman terakhir bisa di-scroll kalau teksnya panjang
